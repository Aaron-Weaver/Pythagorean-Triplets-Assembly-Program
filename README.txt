This is a small program that calculates the pythagorean triplets of the sum that is decided by the user.
The program was made in MIPS Assembly Language

To run this program, I suggest using MARS MIPS Simulator:
http://courses.missouristate.edu/kenvollmar/mars/

or PCSpim:
http://pages.cs.wisc.edu/~larus/spim.html
